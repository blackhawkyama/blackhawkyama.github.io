## How to add a new post

1. Create a file in `_posts/` named `YYYY-MM-DD-some-title.md`.
2. Start it with front matter (the block between the `---` lines above).
3. Write the body in Markdown. Push, and it's live in a minute or two.

## Markdown you'll actually use

Code blocks with syntax highlighting:

```bash
nmap -sC -sV -oN scan.txt 10.10.10.10
```

Inline code like `cat /etc/passwd`, **bold**, _italics_, and links work as
you'd expect.

> Blockquotes are handy for callouts — "the box was rooted via an SUID
> binary" etc.
